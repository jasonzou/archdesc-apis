# Log

## Log configuration

- Edit [api/etc/taxonomy-api.yaml](https://github.com/jasonzou/archdesc-apis/blob/v0.1.1/app/taxonomy/cmd/api/etc/taxonomy-api.yaml)
```
Log:
  ServiceName: taxonomy-api
  Encoding: plain
  Model: Console
  Level: info
```

## In App

```
logx.Info("xxx")
logx.InfoF("xxx", xxx)
logx.Error("xxx")
logx.ErrorF("xxx", xxx)
```
